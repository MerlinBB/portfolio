Audience Trading Platform (London) Ongoing consultation

thefilmhouse.tv

The Film House (Qatar) November/December 2013

UX consultation and front end development for Qatar based film production company, The Film House's portfolio site. The end result is a slick and minimal site, showcasing their amazing images and films. Built from the ground up as a custom WordPress theme, with the help of ACF, creating a simple and intuitive admin for content management. A JavaScript focused (rather than PHP) approach, with lazy loading of content and smooth CSS3 interactions leads to an immersive user experience. A responsive design ensures the content always looks great, and the UI is accesible accross desktop, table and mobile devices. Developed using a modern tooling environment, utilising Bower and Grunt to automate concatenation and minification of assets along with automated jshint and csslint testing. Checkout the The Filmhouse site, or the repo on GitHub.

Chimera

UK2 Group (London) April 2013 — June 2014

Chimera is a unification product, with the end goal of consolidating many different ISP brands back and front ends into a single system, for both admins and end customers. The app is powered by a RESTful JSON API, sitting between a Perl back end and Backbone.js front end. My role in the project was taking responsibility of user experience, visual design, interactions and UI framework. I helped define user requirements, write feature specs, create wireframes and develop full interactive UIs. Using a very modular approach to the UI framework allowed for quick development of new features, while ensuring a consistant user experience and easy maintainability.
